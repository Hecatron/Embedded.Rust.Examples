To setup the needed tools for rust development

## Installation

```sh frame="none"
# First we need the arm target for the stm32
rustup target add thumbv7em-none-eabihf

# Next we need to install binstall for using cargo to do binary installs
cargo install cargo-binstall

# we need probe-rs, this includes flashing and debugging utils
cargo binstall probe-rs
```

## VSCode debugger

  * For VSCode install the extension  
    [Debugger for probe-rs](https://marketplace.visualstudio.com/items?itemName=probe-rs.probe-rs-debugger)
  * For the launch.json file we can use an example like this  
    https://probe.rs/docs/tools/debugger/#using-the-launch-request-type
